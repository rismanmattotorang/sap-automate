Billing Document Creation

Create billing documents from sales orders or deliveries using transaction VF01. Billing types include F2 (invoice), G2 (credit memo), L2 (debit memo), and S1 (cancellation).

The system determines pricing through the pricing procedure assigned to the sales document type; condition records from tables KONP and KONV are evaluated at billing time.

Output determination is configured via transaction NACE; successful billing triggers SD-to-FI integration, posting accounts receivable, revenue, and tax to the appropriate G/L accounts.
