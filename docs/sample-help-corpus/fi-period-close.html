Period-End Close in SAP FI

The period-end close process in SAP FI consists of opening and closing posting periods, executing foreign currency revaluation, posting accruals and deferrals, reconciling document line items between BSEG and FAGLFLEXA, and generating the balance audit trail.

Use transaction code T001B to maintain the table that controls which posting periods are open for each company code and account type.

Foreign currency revaluation is executed via transaction FAGL_FC_VAL; the run revalues open items in foreign currencies at the closing exchange rate and posts the unrealised gains and losses to the configured G/L accounts.

Run reconciliation report FAGLF03 to confirm that the general ledger and document line items are consistent before closing the period.
