Payroll Run Execution

Execute the payroll run using transaction PC00_M99_CALC or country-specific variants. The driver program reads infotypes from tables PA0008 (basic pay), PA0014 (recurring payments), and PA0015 (additional payments).

Payroll results are stored in cluster B2 (table PCL2). Use transaction PC_PAYRESULT to inspect a single employee's wage type results for a payroll period.

Post payroll results to FI/CO via transaction PC00_M99_CIPE; the posting groups wage types into symbolic accounts, which are mapped to G/L accounts in customising.
