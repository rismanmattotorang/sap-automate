Goods Movement Posting

Post goods movements using transaction MIGO. The system supports movement types 101 (goods receipt for a purchase order), 102 (reversal of GR), 122 (return delivery), 261 (goods issue to a cost centre), and 311 (transfer posting).

Each posting updates table MSEG (material document line items), MKPF (material document headers), and the material valuation tables according to the price-control indicator (V for moving-average, S for standard price).

Batch-managed materials require a batch number; serial-managed materials require a serial number. The system enforces stock type consistency (unrestricted, quality inspection, blocked).
